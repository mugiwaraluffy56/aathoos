# Assets

Source design assets for the aathoos brand.

## What Goes Here

- Master app icon (SVG or high-res PNG)
- Logo variants (light, dark, monochrome)
- Brand colors and typography reference
- Shared imagery used across platforms and the website

## Platform-Specific Formats

Platform-specific icon formats are generated from the sources here and live in their respective platform directories:

- `macos/` — `.icns` files
- `windows/` — `.ico` files
- `linux/` — `.png` at required sizes (48x48, 128x128, 256x256, 512x512)
